# Bag of Words Project — Summary

- **Dataset size:** 8 documents
- **Vocabulary size:** 34 unique words
- **Feature vector dimension:** 34
- **Classifier accuracy:** 0.50
- **Labels:** 0 = Negative, 1 = Positive

## How Bag of Words Works

1. Take a collection of text documents.
2. Build a vocabulary of all unique words across all documents.
3. For each document, count how many times each vocabulary word appears.
4. Each document becomes a numerical vector of word counts.
5. These vectors can be used as input to machine learning models.
